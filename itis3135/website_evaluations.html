Website Evaluations

Apple Website

The Apple website is a great example of modern, clean, and effective design.

  • Contrast: Apple uses a strong contrast between text and background, making content easy to read.
  • Repetition: Consistent fonts, spacing, and layout create a uniform experience.
  • Alignment: Everything is aligned properly, creating a sleek and professional design.
  • Proximity: Related content is grouped well, improving readability.

Arngren.net

This website is an example of cluttered and outdated web design.

  • Contrast: Poor color choices make text hard to read.
  • Repetition: Inconsistent fonts and layouts create a messy look.
  • Alignment: Elements are misaligned, making navigation difficult.
  • Proximity: No proper spacing, everything feels cramped.
