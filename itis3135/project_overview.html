Client Project Overview

Project Name

Ahmed Shire Photography Portfolio

Project Overview

  • Function / Purpose: This web application is a professional photography portfolio website for Ahmed Shire. It showcases his photography work and style in a visual, accessible, and organized format.
  • Intended Users: Potential clients, collaborators, photography enthusiasts, and social media followers interested in hiring or viewing Ahmed’s photography.
  • Content Overview: The site will feature a homepage, an about section, a gallery of Ahmed’s work, a contact page, and a services page outlining what he offers.

Site Map

The sitemap below outlines the overall structure of the photography portfolio site we are developing for Ahmed Shire. It illustrates the navigation flow and page hierarchy for a better understanding of user experience.

Page Design

  • Home (index.html)
    Purpose: Showcase featured photos and provide an overview.
    Audience: General visitors.
    Content: Welcome message, hero image, short intro.
    Interactivity: Navigation links, call to action buttons.
  • About (about.html)
    Purpose: Tell Ahmeds story and photography journey.
    Audience: Curious visitors, potential clients.
    Content: Biography, style, background.
    Interactivity: None beyond navigation.
  • Gallery (gallery.html)
    Purpose: Display Ahmeds photos by category.
    Audience: Photography enthusiasts, potential clients.
    Content: PNG and JPG images with captions.
    Interactivity: Clickable thumbnails for lightbox/gallery view using JavaScript.
  • Services (services.html)
    Purpose: Describe services like event, portrait, and street photography.
    Audience: Hiring clients.
    Content: Service descriptions and prices.
    Interactivity: Accordion-style JS sections or tabs.
  • Contact (contact.html)
    Purpose: Allow users to reach out for inquiries.
    Audience: Interested clients.
    Content: Contact form with fields (name, email, message).
    Interactivity: Form validation, dynamic thank-you message.

Dynamic Functionality

  • JavaScript Gallery on gallery.html - opens full-size image previews (inspired by: PhotoSwipe).
  • Contact Form on contact.html with input validation using JavaScript.
  • Accordion Section or Tabbed Services List on services.html (inspired by: jQuery UI Accordion).
